虽然 [ClawSimple](/) 提供了便捷的全托管服务，但您完全可以免费在自己的基础设施上运行 OpenClaw。

本指南将带您一步步在一个全新的 Ubuntu 服务器上搭建自托管的 OpenClaw 实例。

## 准备工作

在开始之前，您需要准备：

1.  **一台全新的 Ubuntu 服务器**：推荐 Ubuntu 22.04 LTS 或更新版本。2核心 vCPU 和 4GB内存的配置是一个不错的起点（例如阿里云、腾讯云、AWS 或 Hetzner）。
2.  **Root 权限**：您需要拥有服务器的 `root` 权限（或 sudo 权限）以便通过 SSH 登录。
3.  **Telegram Bot Token**：通过 Telegram 上的 [@BotFather](https://t.me/BotFather) 创建一个新机器人并获取 API Token。
4.  **Telegram User ID**：找到您的数字 User ID（可以通过 [@userinfobot](https://t.me/userinfobot) 查看），以便将自己加入白名单。
5.  **AI API Key**（可选）：如果您想使用特定的模型，准备好 OpenAI 或 Z.ai 的 API Key。

## 第一步：连接到您的服务器

打开终端并 SSH 登录到您的服务器：

```bash
ssh root@your-server-ip
```

## 第二步：运行安装程序

我们提供了一个自动化的安装脚本，能够处理依赖项（Node.js, pnpm）、创建 systemd 服务以及初始化配置。

将以下命令复制并粘贴到您的终端中：

```bash
curl -fsSL https://clawsimple.com/api/install | bash
```

脚本运行过程中会询问您几个问题：

*   **Telegram Bot Token**：粘贴您从 BotFather 获取的 Token。
*   **Telegram User ID**：输入您的数字 ID。
*   **AI API Keys**（可选）：如果有，请输入；否则可以使用默认设置。

## 第三步：部署完成

脚本运行结束后，您会看到如下成功提示：

```text
==========================================
  ClawSimple Installation Complete!
==========================================
```

您的 OpenClaw 机器人现在已经作为一个名为 `clawsimple` 的 systemd 服务在后台运行了。

### 常用命令

*   **查看状态**：`sudo systemctl status clawsimple`
*   **查看日志**：`sudo journalctl -u clawsimple -f`
*   **重启服务**：`sudo systemctl restart clawsimple`

## 下一步

您的机器人现在应该已经上线了！试着在 Telegram 上给它发送 `Hi` 看看。

由于是自托管，您拥有对配置的完全控制权。主要配置文件位于：
*   环境变量：`/opt/clawsimple/.env`
*   OpenClaw 配置：`/opt/clawsimple/.openclaw/openclaw.json`
