每个人都常梦想拥有一个 Jarvis。但在把你的日历、邮件甚至私密日记交给 AI 之前，你首先需要的是**绝对的隐私**和**控制权**。

这就是为什么我们需要 OpenClaw —— 一个完全运行在你掌控下的开源 AI 代理。（如果你还没有自己的实例，用 [ClawSimple](https://clawsimple.com) 可以在几分钟内拉起一个，但这不是重点）。

今天我们要聊的是**调教**。如何通过一段精心设计的 Prompt，把一个刚部署好的“白板”机器人，训练成一个懂你心意、每天早晚为你服务的私人行政助理。

### 核心理念：不仅是工具，而是“独立个体”

我们建议你不要直接用主力账号授权，而是为它赋予一个独立的身份。

很多用户习惯用自己的 Google 账号直接授权给 Bot，这当然没问题。但为了更专业、更安全，也是为了更有“仪式感”，我们建议你：

**把 Bot 当成一个独立的“人”。**

1.  **为它申请一个专用邮箱**：比如 `assistant.yourname@gmail.com`。
2.  **为它准备一个专用浏览器 Profile**：在 Chrome 或 Edge 中新建一个用户配置，专门登录这个账号。
3.  **赋予它权限**：在你的主日历中把这个账号设为“可以查看”或“可以编辑”，在你的 Notion 团队中邀请这个账号。

这样，当 Bot 需要认证（OAuth）时，你就在它的专用浏览器里完成授权。它就真的像你的数字助理一样，拥有了自己的身份，只访问你授权给它的数据。

### 开始调教：与 Bot 对话

部署好 OpenClaw 并绑定 Telegram 后，直接发给它这段指令。这是我们为你准备好的“咒语”：

> **我需要你帮我做两件事。我可以提供给你 Notion 的 API Key 和 Maton (或其他服务) 的 API Key，并配合你做好网页认证。其余的逻辑请你尽量自己调用 Skill 完成，有困难再问我。**
>
> **Task 1: 早上 7 点「起床 Briefing」**
>
> 每天 **7:11**（ProTip：因为 AI Provider 会处理大量的整点定时任务，所以避开整点高峰，可以保证响应速度和成功率，甚至花费也会更少——因为更容易找到低价供应商）自动发一条消息给我，内容包括：
> 1.  **今天的日程**：读取我的日历 Skill。
> 2.  **今日天气 & 穿衣建议**：调用天气 Skill，告诉我今天是否需要带伞。
> 3.  **昨日未回邮件摘要**：调用邮件 Skill，找出昨天未回复的重要邮件，用摘要 Skill 总结 3 封给我。
> 4.  **行业情报**：利用 Web 搜索 Skill，抓取 3 条我关注行业的最新新闻并摘要。
>
> **Task 2: 晚上「复盘 / 日记机器人」**
>
> 每天晚上 **21:00**，或者当我发送“总结下”的时候：
> 1.  **读取今日聊天记录**：回顾我们今天的所有对话。
> 2.  **生成复盘笔记**：把今天的关键事件、我的心情变化、新产生的 Todo 整理成一篇结构化的日记。
> 3.  **写入笔记**：调用 Notion/Obsidian Skill，把这篇复盘自动写入我的笔记本。

### 接下来会发生什么？

发送完这段话后，OpenClaw 会开始分析你的需求：

1.  **整理需求**：它会列出它需要哪些权限（比如 Google Calendar, Gmail, Notion 等）。
2.  **发起认证**：它会丢给你几个认证链接。
3.  **完成部署**：这时，请打开你为“助理”准备的专用浏览器，点击这些链接完成授权。

认证完成后，它会告诉你：“大功告成！我已经设置好了定时任务。”
