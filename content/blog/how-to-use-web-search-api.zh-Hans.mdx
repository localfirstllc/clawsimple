想要您的 AI 机器人能像真人一样去必应或谷歌搜索最新的新闻、天气或者商品价格吗？

默认情况下，大多数 AI 模型的知识库都是截止到几个月甚至几年前的。但只要给它配置一个“搜索 API”，您的机器人就能瞬间“长出耳朵和眼睛”，随时获取当下的最新资讯！

我们首推使用 **Brave Search API** 来实现这个功能。尽管官方最近更新了规则要求绑定银行卡，但它每个月依然大方地提供 **5美元的免费测试额度**（相当于约 1,000 次搜索请求），这对普通用户来说完全够用。只要按照本文教您的绝招设置“限额”，就绝对不会产生扣费风险。

请参考以下图文教程，只需这简单的 5 个步骤，即使您完全不懂代码也能轻松获取您的专属 API 密钥。

---

## 第一步：访问 Brave 搜索引擎 API 官网

1. 在您的浏览器中，打开 Brave API 的官方网址：[https://brave.com/search/api/](https://brave.com/search/api/)。
2. 在页面中间或右上角，点击注册按钮。

![Brave API 首页](/images/blog/brave-api-home.png)

## 第二步：注册并登录您的账号

1. 在注册页面，输入您的电子邮件地址，并设置一个密码进入注册流程。
2. 勾选同意服务条款，然后点击创建账号。
3. Brave 会向您的邮箱发送一封验证邮件。打开您的邮箱，点击里面的验证链接，即可完成注册并登录到 Brave API 的管理后台。

![Brave 搜索仪表盘](/images/blog/brave-api-dashboard.png)

## 第三步：领取免费搜索额度

Brave 目前对所有用户默认每月赠送 $5 的免费请求额度，但**前提是需要绑定一张银行卡（信用卡或借记卡均可）作为身份验证**。

1. 登录后台后，在左侧菜单中找到并点击 **“Search API” (搜索 API)** 仪表盘。
2. 按照屏幕提示输入您的付款信息完成订阅激活。
3. **⚠️ 防扣费绝招（最关键的一步）**：绑卡成功后，马上在后台菜单找到 **Usage limits（使用限额）**。
4. 将 **Monthly spend limit（每月最高支出上限）** 强制设置为 `$5.00`。
  *(别担心！由于系统每月默认赠送您 5 美元，这里的上限填 5 就意味着“最高只允许系统消耗掉送我的这 5 块钱”。达到 5 美元时服务就会暂停，**绝不会从您的卡里额外扣取任何真实费用**。)*
5. 保存！

![Brave API 使用限额界面](/images/blog/brave-api-usage-limits.png)

## 第四步：生成属于你的 API 密钥 (API Key)

接下来，我们要拿到那把能让 AI 连入互联网的“神奇钥匙”。

1. 在后台菜单的 API Keys 或设置中，寻找 **“Create API Key” (创建 API 密钥)** 或者 **“Generate”** 按钮。
2. 系统会要求给这个密钥随便起个名字，您可以输入例如 `MyOpenClawBot`。
3. 生成成功后，页面上会显示一长串由英文字母和数字组成的无规律代码。这就是我们要找的 API Key（密钥）。
4. 点击这行代码旁边的 **“Copy” (复制)** 按钮，将它保存在您的电脑剪贴板里。

**⚠️ 请注意**：这个密钥就像您的密码一样，请不要发给任何陌生人哦！

![Brave API 密钥界面](/images/blog/brave-api-keys.png)

## 第五步：告诉 OpenClaw 机器人你的 API 密钥

这最后一步最简单，只要把刚刚复制的密钥发给 OpenClaw 机器人，让他帮你配置好就可以了。比如：
`这是我的 Brave API Key，你帮我配置好 web search：` + `你的 API Key`

机器人会自动帮你配置好，然后返回类似以下信息：
```
Gateway restart restart ok (gateway.restart)
Restart to apply Brave Search API key configuration
Run: openclaw doctor --non-interactive
```
如果它说没有成功——重试一次！
