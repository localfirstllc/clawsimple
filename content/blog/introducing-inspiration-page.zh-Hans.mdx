## 我们为什么要搭建这个页面？

一个自主 agent 的真正威力在于它能执行多步工作流、与互联网交互并综合整理信息，而不仅仅是闲聊。

但是，给 agent 写 Prompt 和在聊天框里提问是两码事。一个糟糕的指令可能会让 agent 陷入死循环，或者错误地调用工具。更重要的是，很多用户总是试图通过安装一个新的 skill 来解决所有问题，却不知道**安装未经测试的社区代码是导致 agent 崩溃的头号原因**。

我们推出 Inspiration 页面正是为了解决这些痛点：

**✅ 确保可用的结果：** Inspiration 页面上的每一个 Prompt 都经过了严格的测试。您可以确信，只要把它粘贴到您的 Telegram bot 里，就能直接获得稳定、可用的结果。

**✅ 零配置风险：** 您不需要安装任何额外的 skills 或修改部署配置。这些用例完全基于 OpenClaw 核心、稳定的功能集。

**✅ 社区认可：** 我们不会发布一堆中看不中用的理论概念。我们只会发布并持续更新那些真正有效、且受到社区热烈欢迎的实际应用案例。

## 开箱即用，随时上手

无论您使用的是 **BYOM**、**Standard** 还是 **Max** 订阅方案，所有精选的 Inspiration 用例都支持开箱即用。

您的 agent 已经具备了强大的内置能力：它可以执行复杂的网络搜索、读取和分析网页内容、或者处理长篇文档。Inspiration 页面只是为您提供了最精确的“话术”，帮您高效地解锁这些能力。

例如，与其模糊地让机器人去“搜一下新闻”，您可以从 Inspiration 页面复制一段结构化的 Prompt，指示 agent 去搜索特定来源、提取关键数据，并将其整理成一份完美的晨间简报。
