AIアシスタントに、生身の人間のように最新のニュース、天気、商品の価格を検索してほしいと思ったことはありませんか？

初期設定のままでは、AIモデルの知識は数ヶ月〜数年前で止まっていることがあります。ですが、「検索API」を設定してあげれば、あなたのロボットは「目と耳」を持ち、いつでも最新情報を得られるようになります！

これを実現するために最もオススメなのは **Brave Search API** の利用です。最近のルール変更でクレジットカードの登録が必要になりましたが、今でも**月額$5の無料テスト枠**（毎月約1,000回の検索が可能）という太っ腹なプランを提供しており、一般ユーザーならこれで十分すぎます。この記事で紹介する「裏ワザ」で利用上限を設定しておけば、勝手に課金されてしまうリスクは絶対にありません。

コードが全くわからなくても大丈夫！以下の5つの簡単なステップに従って、API Keyの取得から設定までを完了させましょう。

---

## ステップ 1：Brave Search APIの公式サイトを開く

1. ブラウザで Brave API の公式サイトにアクセスします：[https://brave.com/search/api/](https://brave.com/search/api/)
2. ページの中央または右上にある、登録ボタン（Sign Up / Get started）をクリックします。

![Brave API 首页](/images/blog/brave-api-home.png)

## ステップ 2：アカウント登録とログイン

1. 登録画面でメールアドレスを入力し、パスワードを設定して登録を進めます。
2. 利用規約に同意してアカウントを作成します。
3. Braveから確認メールが届きます。メールを開いて認証リンクをクリックすると登録が完了し、Brave API の管理画面（ダッシュボード）にログインできます。

![Brave 搜索仪表盘](/images/blog/brave-api-dashboard.png)

## ステップ 3：無料の検索枠をもらう

Braveは、全ユーザーに対して毎月デフォルトで$5の無料リクエスト枠を付与します。ただし、**本人確認としてクレジットカード（またはデビットカード）の登録が必須**となっています。

1. ログイン後、左側のメニューから **Search API** のダッシュボードをクリックして開きます。
2. 画面の指示に従い、支払情報を入力してサブスクリプションをアクティブにします。
3. **⚠️ 課金防止の裏ワザ（最も重要なステップ）**：カードの登録が完了したら、すぐに左側メニューで **Usage limits（使用制限）** を探してクリックします。
4. **Monthly spend limit（月間最大支出上限）** を強制的に `$5.00` に設定します。
  *(心配しないで！システムから毎月$5が無料でプレゼントされるため、ここで上限を$5にするということは、「貰った無料分の$5だけを消費することを許可する」という意味になります。$5に達した時点でサービスが一時停止し、**実際のクレジットカードからは絶対に何も請求されません**。)*
5. 保存（Save）をクリック！

![Brave API 订阅界面](/images/blog/brave-api-usage-limits.png)

## ステップ 4：あなたのAPI Key（鍵）を作成する

次に、AIをネットに繋ぐための「魔法の鍵」を取得します。

1. ダッシュボードのメニューから API Keys または設定へ進み、**Create API Key** または **Generate** ボタンを探します。
2. この鍵に適当な名前を付けます（例：`MyOpenClawBot`）。
3. 生成されると、英数字がランダムに並んだ長い文字列が表示されます。これがAPI Key（鍵）です。
4. 文字列の横にある **Copy（コピー）** ボタンをクリックして、クリップボードに保存します。

**⚠️ 注意**：この鍵はパスワードと同じくらい重要なものです。絶対に他人には教えないでください！

![Brave API 密钥界面](/images/blog/brave-api-keys.png)

## ステップ 5：OpenClawロボットにAPI Keyを教える

最後のステップが一番簡単です。先ほどコピーした鍵を、OpenClawロボットのチャット画面で直接送信し、設定を依頼しましょう。例えばこのように送ります：
`これが私のBrave API Keyです。Web検索の設定をお願いします：` + `あなたのAPI Key`

ロボットが自動的に設定を行い、以下のようなメッセージを返します：
```
Gateway restart restart ok (gateway.restart)
Restart to apply Brave Search API key configuration
Run: openclaw doctor --non-interactive
```
もし失敗したと返ってきたら、もう一度試してみてください！

## 実際にテストしてみましょう！

設定完了です！Telegram を開いて、ロボットとのチャット画面に戻ってください。

最近の出来事や、ネットに繋がないとわからない質問をしてみましょう。例えば：
* *「今日のAppleの株価を見てくれない？」*
* *「過去24時間のAIに関する主要なニュースを要約して。」*

ロボットがスムーズに答えるだけでなく、回答の最後に参考にしたWebサイトのリンクも貼ってくれることに驚くはずです。
